El desarrollo web ha crecido muchísimo, voy a tratar de darte una introducción muy básica, pero que te dé las bases que necesitas para iniciar.

Internet está lleno de páginas webs, tanto de contenido estático como dinámico, una web con contenido estático es sólo un sitio que rendera contenido que puede ser textos, imágenes, vídeos, etc. Las webs con contenido dinámico son mucho más interesantes, debido a que agregan algo más de complejidad en su estructura, generalmente este tipo de páginas hacen lo mismo que las estáticas pero su contenido puede cambiar, el contenido es leído de alguna base de datos, es decir, está almacenado en algún sitio, entonces si el contenido cambia, ese cambio se verá reflejado en el sitio web, detrás de este proceso hay más cosas, pero las vamos a ir viendo en próximas entradas.

Para entrar a un sitio web usas un navegador web, el más usado es Chrome que es de Google, pero existen otros como Safari, Firefox, Edge, etc.

Cuando escribes una dirección web (dominio) en la barra de navegación, el navegador por medio del protocolo HTTP realiza una petición al servidor donde está alojado el sitio al que intentas acceder.


> HTTP, de sus siglas en inglés: "Hypertext Transfer Protocol", es el nombre de un      protocolo el cual nos permite realizar una petición de datos y recursos, como pueden ser documentos HTML. Es la base de cualquier intercambio de datos en la Web, y un protocolo de estructura cliente-servidor, esto quiere decir que una petición de datos es iniciada por el elemento que recibirá los datos (el cliente), normalmente un navegador Web.

> fuente: [MDN web docs](https://developer.mozilla.org/es/docs/Web/HTTP/Overview)


El servidor recibe la petición, la lee y genera una respuesta, esta respuesta llega al navegador y entonces se te carga la página web que pediste.

Este tipo de comunicación cliente servidor es la forma más común de intercambio de datos en la web.

Para iniciar a crear sitios webs estáticos o dinámicos lo primero que hay que aprender es HTML, CSS y Javascript. HTML es un lenguaje de marcado basado en etiquetas, que nos sirve para crear documentos de páginas webs, con CSS podemos darle estilos a nuestro documento HTML y con Javascript podemos agregar interactividad en nuestro sitio, es decir, que ciertos elementos HTML puedan reaccionar cuando el usuario haga click sobre ellos.
